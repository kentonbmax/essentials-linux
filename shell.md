- Shell Commands!
-- Alias
1. Create alias: `alias la='ls -l -a --color=auto'`

-- File Commands
1. More info: `ls -F`
1. Show hidden: `ls -a`
1. Long Listing auto color: `ls -l -a --color=auto`

-- Dir Commands
1. Create Dir: `mkdir mydir`

-- Misc
1. Last Arg: `!$`

-- Functions
1. Set Function: `function set {}`
